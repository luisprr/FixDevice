# FixDevice – Plataforma de Servicios Técnicos

FixDevice es una página web diseñada para conectar a usuarios con técnicos especializados en la reparación de dispositivos electrónicos como computadoras, laptops y celulares. Este proyecto presenta una landing page moderna, responsiva y enfocada en la experiencia del usuario.

---

## Características principales

- Diseño moderno y responsivo.
- Menú hamburguesa con overlay para móviles.
- Secciones completas: Inicio, Servicios, Quiénes Somos, Planes, Testimonios y Contacto.
- Tipografías desde Google Fonts (Saira Condensed y Poppins).
- Estilos personalizados en `assets/styles/styles.css`.
- Comportamientos interactivos desde `assets/scripts/main.js`.

## Cómo Ejecutar el Proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/luisprr/FixDevice.git
```

2. Ingresar al directorio:

```bash
cd FixDevice
```

3. Abrir el archivo principal:

```bash
index.html
```
